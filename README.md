# 🔢 Multiplication Table Generator

An interactive Streamlit app that helps users quickly generate multiplication tables with ease.
Designed to make learning multiplication simple, fast, and engaging.

## ✨ Features

| Feature            | Description                                               |
| ------------------ | --------------------------------------------------------- |
| 🔢 Table Generator | Generate multiplication tables for any number             |
| ⚡ Instant Results  | Get tables instantly with real-time updates               |
| 🎯 Easy to Use     | Simple and beginner-friendly interface                    |
| 📚 Learning Tool   | Helps students practice and improve multiplication skills |
| 🌐 Web-Based       | Accessible from any browser using Streamlit               |

---

## 🛠️ Tech Stack

| Technology       | Purpose                              |
| ---------------- | ------------------------------------ |
| 🐍 **Python**    | Core programming language            |
| 🎈 **Streamlit** | Builds the interactive web interface |

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/KrishBharadwaj5678/MultiplicationTableGenerator.git
```

### 2️⃣ Navigate to the Project

```bash
cd MultiplicationTableGenerator
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```
